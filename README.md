# JackBotTV

Simple Python application that can play basic JackBox Party Pack games using Selenium

Supported Games:
* JackBox Party Pack 1
  * Fibbage XL
  
* JackBox Party Pack 2
  * Quiplash XL
  * Fibbage 2

* JackBox Party Pack 3
  * Quiplash 2
  * Guesspionage

* JackBox Party Pack 7
  * Quiplash 3
